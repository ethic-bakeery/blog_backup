# IMAGE AND VIDEO VERIFICATION

[bit.ly/bcattools](http://bit.ly/bcattools) | Last update: August 2023

## Tools

### üîç Reverse Image Search

#### [InVID-WeVerify](https://www.weverify.eu/invid/)
Verification plugin to help journalists verify images and videos. It provides contextual data, metadata, reverse search (Google, Yandex, Baidu), image forensic, and magnifier tools.
- **Free**: Yes
- **Guide**: [Guide To Using Reverse Image Search For Investigations](https://medium.com/)

#### [Google Lens / Desktop](https://www.google.com/search/about/)
Google Lens in your browser; it's better than Google Image reverse search.
- **Free**: Yes

#### [VGG Image Search Engine (VISE)](https://www.robots.ox.ac.uk/~vgg/software/vise/)
This standalone application can be used to do reverse image searches on large collections of images.
- **Free**: Yes

#### [RootAbout](https://www.rootabout.com)
Reverse search images on the Internet Archive.
- **Free**: Yes

#### [KarmaDecay](https://karmadecay.com)
Reverse search images on Reddit.
- **Free**: Yes

### üåû Shadow Analysis

#### [SunCalc.org](https://www.suncalc.org)
Make an approximation of the time of the day using shadow direction.
- **Free**: Yes
- **Guide**: [Using the Sun and Shadows for Geolocation](https://suncalc.org/en)

### üñºÔ∏è Metadata Tools

#### [metadata2go](https://www.metadata2go.com)
Check metadata for both photos and videos online.
- **Free**: Yes

#### [Reveal Image Verification Assistant](https://revealproject.eu/)
Forensic tool providing eight filters to detect still image alterations. Available as a web-based tool and also within the InVID verification plugin.
- **Free**: Yes

#### [ExifPurge](https://www.exifpurge.com)
A small portable application to remove EXIF metadata from multiple images at once.
- **Free**: Yes

#### [FotoForensics](https://fotoforensics.com)
Image forensics tool.
- **Free**: Yes

#### [Forensically Image Forensics](https://29a.ch/photo-forensics/)
Web-based image forensics tool. It can easily identify fake or doctored images.
- **Free**: Yes

#### [Irfanview](https://www.irfanview.com)
Windows-based software to extract metadata.
- **Free**: Yes

#### [SpiderPig](https://github.com/QuLogic/spiderpig)
GitHub tool to extract metadata. Command line interface and scriptable.
- **Free**: Yes

#### [Jimpl](https://exif.tools)
Online EXIF data viewer.
- **Free**: Yes

#### [jhead](https://www.sentex.ca/~mwandel/jhead/)
Remove EXIF/metadata from photos.
- **Free**: Yes

#### [Splunk](https://www.splunk.com)
Extract metadata with report-grade analysis and presentation. Not simple to set up.
- **Free**: Yes

### üë§ Facial Recognition

#### [VGG Face Finder (VFF) Engine](https://www.robots.ox.ac.uk/~vgg/software/vff/)
A web application that performs searches for faces over a user-defined image dataset.
- **Free**: Yes

#### [PimEyes](https://pimeyes.com/en)
Face Search Engine and Reverse Image Search.
- **Free**: Partially

#### [Search4Faces](https://search4faces.com)
Search a database of 84 million TikTok accounts.
- **Free**: Yes

#### [Amazon Rekognition](https://aws.amazon.com/rekognition/)
Check how similar two faces are.
- **Free**: Yes
